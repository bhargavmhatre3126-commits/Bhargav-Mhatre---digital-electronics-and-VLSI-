Digital Voting Machine using Verilog
Overview
This project implements a simple digital voting machine using Verilog HDL.
The design counts votes for three candidates and verifies the results through simulation.

Features
Vote counting for 3 candidates
Reset functionality
Clock-based sequential logic
Simulation waveform verification
Tools Used
Verilog HDL
EDA Playground
GTKWave
File Structure
Digital-Voting-Machine │ voting_machine.v # Main voting machine design ├ voting_tb.v # Testbench for simulation ├ waveform.png # Simulation waveform result └ README.md # Project documentation

Simulation
The testbench simulates button presses for each candidate and verifies that the vote counters increment correctly.
